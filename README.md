# 🚀 Formação Full Stack - Rocketseat

Este é o meu espaço de projetos, criados durante a jornada Full Stack, que explora desde o básico até as melhores práticas no desenvolvimento web.

## 🧠 Ferramentas utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- React

## 📂 Estrutura do projeto

- **html/** → Estruturação de páginas
- **css/** → Estilização e layout
- **javascript/** → Lógica e manipulação do DOM
- **node/** → Backend e construção de APIs
- **react/** → Interfaces modernas com componentes
- **desafios/** → Exercícios desenvolvidos de forma independente

## 🎯 Objetivo

Consolidar conhecimentos em desenvolvimento web, evoluindo do basico até aplicações mais estruturadas, com integração entre front-end e back-end.

## 📌 Observação

Os projetos foram construídos principalmente junto com a formação, focando no aprendizado colocando a mão na massa e usando o que foi ensinado.

## 📈 Evolução
Este repositório mostra o meu avanço na trilha Full Stack, englobando tanto os conceitos iniciais quanto os projetos com maior nível de detalhe.
